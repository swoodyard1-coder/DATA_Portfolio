# Manufacturing Performance Dashboard

This project was created in Power BI using the Microsoft Azure Predictive Maintenance dataset.

The dashboard analyzes:

* Machine failures
* Machine models
* Vibration and pressure trends
* Manufacturing performance over time

## Tools Used

* Power BI
* Power Query
* Data Visualization
* GitHub

## Dashboard Features

* Line chart showing vibration trends over time
* KPI cards displaying machine metrics
* Clustered bar chart for failure analysis
* Pie chart showing failure type distribution
* Interactive slicers for machine ID, model, and date filtering

## Business Questions Answered

* Which machine models experience the most failures?
* What failure types occur most frequently?
* How do machine performance metrics change over time?
* How can predictive maintenance improve manufacturing efficiency?
